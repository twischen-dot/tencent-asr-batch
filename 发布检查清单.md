## 📝 发布步骤

1. **初始化 Git 仓库**
   ```bash
   cd github-release
   git init
   git add .
   git commit -m "Initial commit: Tencent ASR batch transcription tool"
   ```

2. **创建 GitHub 仓库**
   - 访问 https://github.com/new
   - 创建新仓库（建议名称：`tencent-asr-batch`）

3. **推送代码**
   ```bash
   git remote add origin https://github.com/your-username/tencent-asr-batch.git
   git branch -M main
   git push -u origin main
   ```

4. **添加仓库描述**
   - 描述：利用腾讯云免费额度批量转写音频文件的工具
   - 标签：`python`, `tencent-cloud`, `asr`, `speech-to-text`, `audio-transcription`

5. **创建 Release**
   - 添加版本号（如 v1.0.0）
   - 添加发布说明
